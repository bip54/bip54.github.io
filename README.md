# bip54.lol

A collection of objections to BIP54 (Consensus Cleanup) from across the bitcoin community.

Static site hosted on GitHub Pages.

## Add a post

Create an issue with a link to the X or Nostr post objecting to BIP54 and it will be added.

## Structure

- `index.html`, `style.css`, `app.js` ‚Äî the site
- `tweets.json` ‚Äî the posts shown (`name`, `handle`, `platform`, `avatar`, `date`, `text`, `url`)
- `img/` ‚Äî locally hosted profile pictures
- `og.svg` / `og-image.png` ‚Äî social share card
